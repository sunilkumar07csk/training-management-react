# Training Management React Application

This is a frontend-only Training Management application built using React functional components. It manages Subjects, Courses, Batches, and Students with proper validations, localStorage persistence, and a simple dashboard. The focus is on React fundamentals, state handling, and UI clarity

## Features
- Subject creation with duplicate prevention
- Course creation with multiple subjects
- Batch scheduling with time validation
- Student assignment with dependent dropdowns
- Dashboard summary
- Dark mode
- UI animations
- No backend required

## Tech Stack
- React (Hooks)
- JavaScript
- CSS

## Assumptions
- Data is stored using browser localStorage
- Application is frontend-only for demonstration
- UI focused for interview evaluation# training-management-react
