# Training Management React Application

This project is a frontend-only Training Management system built using React functional components. It includes Subject, Course, Batch, and Student management modules with validations, localStorage persistence, and responsive UI.

## Features
- Subject creation with duplicate prevention
- Course creation with multiple subjects
- Batch scheduling with time validation
- Student assignment with dependent dropdowns
- Dashboard summary
- Dark mode
- UI animations
- No backend required

## Tech Stack
- React (Hooks)
- JavaScript
- CSS

## Assumptions
- Data is stored using browser localStorage
- Application is frontend-only for demonstration
- UI focused for interview evaluation# training-management-react
